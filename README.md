# OmniSearch

A simple static prototype of **OmniSearch**: a personal AI chat-history search dashboard.

## What this includes
- One-page dashboard UI using only:
  - `index.html`
  - `styles.css`
  - `script.js`
- Mock conversation data from:
  - ChatGPT
  - Claude
  - Gemini
  - Perplexity
  - Copilot
  - Grok
- Search input for keywords/topics/project names.
- Source filters and topic/project filters.
- Results cards that show:
  - AI source
  - conversation title
  - date
  - matched keyword/topic
  - message snippet
  - tags
  - relevance score
  - “Open original chat” placeholder button
- Extra dashboard sections:
  - Search
  - Sources
  - Imports
  - Saved Results
  - Settings

## Beginner-friendly run instructions
1. Download or clone this folder.
2. Make sure these files are in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Open `index.html` in your browser (double-click it).
4. Start typing in the big search bar.
5. Click source/topic filters to narrow results.

## Notes
- This is a **front-end only** prototype.
- There is **no backend**, **no login/auth**, and **no external APIs**.
- All data is static mock data in `script.js`.
